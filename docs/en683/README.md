## EN683 NPU Model Zoo

EN683 features a dedicated NPU optimized for real-time, edge inference without requiring a cloud connection.    
It supports fast INT8 inference across a wide range of AI tasks:

- **Vision Recognition:** Classification, Object Detection, Face Recognition

- **Facial Analysis:** Facial Landmark Detection, Head Pose Estimation

- **Human Pose & Segmentation:** Pose Estimation, Instance Segmentation

For detailed specifications, supported models, operations, and demo videos, refer to the [EN683 Overview](https://resonant-duke-420.notion.site/EN683-NPU-SDK-for-AI-21482f944b8680eb9af2dac75f51d186).
