## EN675 NPU Model Zoo
**EN675** features a dedicated NPU optimized for real-time inference on edge devices, without requiring a cloud connection. It supports fast, lightweight INT8 inference and enables core AI tasks such as classification and object detection. For detailed specifications, supported models, operations, and demo videos, refer to the [EN675 Overview](https://resonant-duke-420.notion.site/EN675-AI-NPU-Solution-d407c17992d8447b9c98ac2bfede8cdb).
